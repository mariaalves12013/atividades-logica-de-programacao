📚 Exercícios de Lógica com JavaScript

Este repositório contém uma série de exercícios básicos de lógica de programação utilizando JavaScript, desenvolvidos para praticar conceitos fundamentais da linguagem e interação com o usuário no navegador.

Os exercícios utilizam recursos simples como:

prompt() para entrada de dados

alert() e console.log() para saída de informações

Estruturas condicionais (if, else)

Operadores matemáticos e lógicos

Todos os exercícios foram implementados em arquivos HTML simples com JavaScript embutido.

🧠 Conteúdos Praticados

Durante os exercícios são trabalhados conceitos essenciais de programação:

Variáveis

Entrada de dados

Conversão de tipos (Number())

Operações matemáticas

Estruturas condicionais

Operadores lógicos

Manipulação de strings
